
# Nuna - Tienda Online

Nuna es una tienda online de productos andinos sostenibles, textiles de alpaca y gadgets inteligentes. Este proyecto busca fusionar la tradición andina con la tecnología contemporánea, ofreciendo una experiencia de compra única, ética y moderna.

## Características principales

- Catálogo de productos premium: textiles, gadgets y más.
- Diseño responsivo y moderno (Tailwind CSS).
- Experiencia de usuario optimizada y accesible.
- Carrito de compras interactivo.
- Código modular y fácil de mantener.

## Estructura del proyecto

- `index.html`: Página principal de la tienda.
- `styles.css`: Estilos personalizados y paleta andina.
- `script.js`: Lógica de la tienda y gestión de productos.

## ¿Cómo usar?

1. Clona este repositorio:
	```bash
	git clone https://github.com/EzerZuniga/Nuna-online.git
	```
2. Abre el archivo `index.html` en tu navegador favorito.
3. ¡Explora, agrega productos al carrito y disfruta la experiencia Nuna!

## Tecnologías utilizadas

- HTML5, CSS3, JavaScript
- Tailwind CSS
- Google Fonts (Cormorant, Jost)
